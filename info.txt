class router
	два свойства:
		таблица маршрутов routes
		текущий маршрут route
	Методы:
		метод add:(регулярное выражение, текущий маршрут) - Заполняет 														Таблицу маршрутов self::$routes
		getRoutes - возвращает таблицу маршрутов
		getRoute - возвращает содержимое свойства route 
		Метод matchRoute(текущий маршрут) - Ищет совпадение с запросом в 														таблице маршрутов
		Метод dispatch - перенаправляет URL по корректному маршруту на основе текущего маршрута
class view на основе свойств route view layout находит путь к виду и шаблону с помощью метода render
